WEBSITE LAYOUT

Minimal layout design

The single page design is minimal with the Bubble logo in the top left corner and various bubble links to other websites. Much like Reddit, with the Reddit enhancement suite, users are able to click on each bubble to pop open a link to the page. Users will get a very minimal preview of what the bubble link contains however each bubble has a colour corresponding to what the link is - Photo, video, article or advertisement.

Single page web design

Bubble will consist of a single page where a set number of bubble links will appear. Implementation of a timeed bubble reset will disincentivise infinite scroll allowing the user to check the page at a much slower pace.

For the colours of the bubbles - Each colour (Pink, yellow, green, blue) will corresond to what data the link will contain. These are Videos, Articles, Images and advertisments.
